# Via Occulta

**Via Occulta** is a 2D top-down online JRPG / MMO-lite concept inspired by classic SNES-era party RPG structure: tile-based exploration, shared towns, instanced dungeons, separate side-view soft-ATB battles, guild economies, crafting, and world-restoration projects.

## Core Direction

> **Online FF4-style structure + guild economy + instanced dungeons.**

This project explores a game format that preserves the feel of a classic authored JRPG while adding online social systems carefully enough to avoid the chaos and production burden of a full conventional MMORPG.

## Design Pillars

1. **Classic JRPG readability** — clear tile maps, readable sprites, clean menus, and separate battle presentation.
2. **Online without chaos** — shared towns and guild spaces, but private party-based dungeons and story scenes.
3. **Party combat is sacred** — roles, healing, buffs, debuffs, elemental weaknesses, and boss phases matter.
4. **Guilds shape the world** — players build airships, restore shrines, reopen trade routes, and unlock crafting tiers.
5. **Story still matters** — the game keeps kingdoms, crystals, ruins, betrayals, airships, summons, and lunar mysteries.

## First Prototype Goal

Build a vertical slice that tests whether FF4-style side-view party combat remains fun when one or more real players control party members online.

Minimum loop:

1. Two players meet in a shared town.
2. They form a party.
3. Empty party slots are filled by NPC companions.
4. They enter an instanced dungeon.
5. Exploration is top-down and tile-based.
6. Combat switches to side-view soft-ATB battles.
7. They defeat a boss.
8. They return to town with materials.
9. They contribute materials to a guild/town project.
10. The project unlocks new crafting or progression.
